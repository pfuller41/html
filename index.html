Introduction to Programming

Components of the Web

The major pieces are you, your computer running a web browser, the Internet, and servers. Your browser makes requests via the Internet to servers.
These requests are using a protocol called HTTP. And the servers respond with files that the browser displays.
Servers are computers just like yours except they are optimized for sitting in a closet and hosting files
The protocol HTTP is just a very simple protocol that your browser uses to communicate with these servers.

HTML

HTML(Hypertext Markup Language) documents form the majority of content on the web.
HTML contains text content "what you see" and markup "how it looks."

Tags and Elements

HTML elements are what we use to tell a web browser how to display content on a web page.
Most HTML elements consist of an opening and a closing tag with some content between them.
Some elements accept additional values called attributes in their opening tags.
Attributes provide additional information to the browser. For example, the anchor tag takes an attribute called href that defines a URL to link to.

It's Okay to Forget

New programmers often struggle with this because in other learning environments it's often important that you remember a lot of facts.
But programming doesn't work like that! There's just too many facts to remember them all.
Instead, you need to understand a few big ideas and know how to get the details you need when you need them

HTML Document Structure

A part of writing good code is structuring it well.
In HTML particularly, the indentation level increases when an idea begins and decreases when an idea is finished.
The first line should appear on every webpage you make. DOCTYPE html defines the type of the document so.
The html tag opens, our document. Everything that shows on the webpage is placed within this tag.
The head tag defines metadata for our web page. This includes the title of the page (this is what appears in the top of your tab or browser window).
The title tag is closed on the same line it was opened which we usually do if the content of a tag is short. The head tag is closed on a different line.
Notice that inside the body tags you have the content of the web page.
You can see an h tag that is indented just inside the body as well as a p tag that is on the same level of indentation as the h.
